# CipherGuard Engagement Template

Reusable documentation templates used across every CipherGuard Security
Consulting engagement (web, network, Active Directory, mobile, cloud, and
red team operations).

## Contents

- **Client-Brief-Template.md** — the scoping document used to kick off
  every engagement: client background, objective, scope, rules of
  engagement, and deliverables.
- **Deliverables-Checklist.md** — the closing checklist every engagement
  must satisfy before it's considered complete (reports, evidence,
  CVSS/risk scoring, remediation plan, presentation, etc).

## How to use

1. Copy `Client-Brief-Template.md` into the new engagement repo as
   `Client-Brief.md` and fill it in before any testing begins.
2. Use `Deliverables-Checklist.md` to verify the engagement is complete
   before marking it closed.
