# Deliverables Checklist

An engagement is only considered complete when every item below is checked.

- [ ] GitHub repository, cleanly organized
- [ ] Professional README
- [ ] Client-Brief.md (scope, objective, rules of engagement)
- [ ] Methodology section
- [ ] Screenshots / evidence, organized and referenced from findings
- [ ] Executive Summary (non-technical, for a client stakeholder)
- [ ] Technical Report (detailed findings, reproduction steps)
- [ ] CVSS scoring per finding
- [ ] Risk Matrix
- [ ] Remediation Plan
- [ ] 10-minute self-recorded presentation of the engagement
- [ ] LinkedIn post summarizing the engagement
